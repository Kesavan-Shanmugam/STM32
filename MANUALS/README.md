# Reference Manuals & Datasheets (`MANUALS`)

This directory contains the essential documentation required for bare-metal programming of the STM32F4x series.

## Contents
- **DATA SHEET/**: Contains the electrical specifications, memory map, and characteristics of the microcontroller.
- **USER MANUAL/**: Contains the Reference Manual detailing the register-level configuration of every peripheral.
- **STM32F446RE-pinout.png**: Quick visual reference for the MCU/Board pin layout.

Always refer to the Reference Manual (User Manual) for finding out which registers to configure for specific peripheral operations.
